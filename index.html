Vector Concepts in Linear Algebra

Let's break down the concepts of subspace, basis, dimension, and span using the vector:

v = T(3, 4) in the context of R2.

1. Subspace

A subspace refers to a set of vectors that is closed under vector addition and scalar multiplication and includes the zero vector. For a single vector in R2, the subspace it generates is a line through the origin in the direction of that vector.

Subspace Generated by v: The subspace generated by the vector v = T(3, 4) is all scalar multiples of this vector. This is a line in R2 passing through the origin and the point (3, 4).

Mathematically, this subspace can be expressed as:

span{ (3, 4) } = { c * (3, 4) | c ∈ R }

2. Basis

A basis for a subspace is a set of vectors that spans the subspace and is linearly independent. For a single vector v, the basis for the subspace it generates is simply { v }.

Basis for the Subspace: The basis for the subspace generated by v = T(3, 4) is:

{ (3, 4) }

3. Dimension

The dimension of a subspace is the number of vectors in its basis. For a line through the origin in R2 (which is the subspace generated by a single non-zero vector), the dimension is 1.

Dimension: The dimension of the subspace generated by v is 1, as it is spanned by a single vector.

4. Span

The span of a set of vectors is the set of all possible linear combinations of those vectors. For a single vector v, its span is the line in the vector space that includes all scalar multiples of that vector.

Span of v: The span of v = T(3, 4) is:

span{ (3, 4) } = { c * (3, 4) | c ∈ R }

This is a line through the origin in the direction of the vector T(3, 4).

Summary

  • Subspace: The line through the origin in R2 along the direction of T(3, 4).
  • Basis: { (3, 4) }
  • Dimension: 1
